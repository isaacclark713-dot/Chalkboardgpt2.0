# Call Sheet — Coach Brain v14.22

GitHub Pages package.

## v14.22 changes

- Deduplicates generated playbooks by exact Formation + Play.
- Adds stronger final formation-mix enforcement after generation.
- Formation style sliders now work as output targets, aiming within about 4% of the selected slider mix when the play inventory supports it.
- Apply System Preset now sets the page sliders directly to the profile preset instead of leaving default slider bleed.
- Existing system/personal/personnel gates and long-yardage call safeguards are preserved.

## Deploy

Upload these files to your GitHub Pages repository:

- `index.html`
- `global_unique_plays.csv`
- `.nojekyll`

Then hard refresh your phone browser and regenerate the base playbook once.
